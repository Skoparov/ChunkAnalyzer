Реализовано на основе клиент - сервера. 

Параметры для запуска анализатора(сервера) по порядку:
-Имя файла для вывода (опционально, если нет, будет выводиться в станд. вывод)
-Порт (опционально, если нет, использует 7777)

Параметры для запуска загрузчика (клиент) по порядку:
-Адрес сервера
-Порт
-Директории одна за другой
